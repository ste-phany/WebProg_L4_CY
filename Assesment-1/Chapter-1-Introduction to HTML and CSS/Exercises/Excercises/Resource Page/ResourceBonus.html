HTML & CSS Quick Reference Guide

A simple guide to topics we have covered in HTML tags and CSS properties.

HTML Tags

<h1> - <h6>: Headers

Headers define different levels of headings. 

<h1>Main Heading</h1>
<h2>Subheading</h2>
<h3>Smaller Heading</h3>

<p>: Paragraph

Used to define paragraphs of text. 

<p>This is a paragraph.</p>

<a>: Links

Creates hyperlinks to other pages or sections. 

<a href="https://example.com">Visit Example</a>

Absolute Link

link to different websites. 

<a href="https://ph.pinterest.com/">Pinterest</a>

Relative Link

Link To Pages in Your Site 

<a href"index.html">Home</a>

<img>: Image

Displays an image.

<img src="image.jpg" alt="Description">

<ul>, <ol>, <li>: Lists

Creates unordered and ordered lists.

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

<ol>
    <li>First</li>
    <li>Second</li>
</ol>

HTML5 Structural Elements

Three Tags Define an HTML element: start, end, and the contents whithin these tags.

1. Header Element

Contains the web page or document's headings. 

<header></header>

2. nav Element

Contains the main web page navigations. 

<nav></nav>

3. Main Element

Contains the web page main documents. 

<main></main>

4. Footer Element

Contains the web page documents footer. 

<footer></footer>

<div></div> Element

<div> tag defines a division or a section in an HTML document

<div>
    <p> hello world </p>
</div>

CSS Properties

color

Sets the color of text.

p {
    color: blue;
}

background-color

Sets the background color of an element.

body {
    background-color: lightgray;
}

font-size

Sets the size of the text.

h1 {
    font-size: 24px;
}

margin & padding

Controls the spacing outside and inside elements.

div {
    margin: 10px;
    padding: 15px;
}

<hr> Horizontal Rule Element

Configures a horizontal line. 

<p>Hello World</p>
<hr>

Output -

Hello World

Rounded Corners

Configures the horizontal radius and vertical radius of the corner.

h1 {
    border-radius: 15px;
}

Box-Shadow Property

Configure the horizontal offset, vertical offset, blur radius, and vlaid color value.

h1 {
    font-size: 36px;
    box-shadow: 5px 5px 5px #828282;
}

Opacity Property

Configure the opacity of an element.

h1 {
    background-color: #FFFFFF;
    opacity: : 0.3;
}

Gradients

A smooth blending of shades from one color to another.

h1 {
    background-color: #8FA5CE;
    background-image:  linear-gradient(#FFFFFF, #8FA5CE); 
}

Padding Property Shorthand: Two Values

Each values configures top, bottom padding & left, right padding.

h1 {
    border: 2px solid #ff0000; 
    background-color: #cccccc;
    padding: 20px 10px;
}

Padding Property Shorthand: Four Values

Configure top, right, bottom, and left padding.

h1 {
    border: 2px solid #ff0000;
    width: 250px; 
    background-color: #cccccc;
    padding: 30px 10px 5px 20px;
}
